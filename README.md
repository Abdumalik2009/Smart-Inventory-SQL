# Smart Inventory System (SQL Server)

Это мой второй проект, в котором я реализовал автоматизацию склада.

### Что здесь особенного:
*   **Автоматические скидки:** Система сама снижает цену на 10%, если товара слишком много.
*   **История цен:** Каждое изменение фиксируется в отдельной таблице.
*   **Контроль данных:** Использованы продвинутые триггеры SQL Server.

**Технологии:** T-SQL, SQL Server Management Studio.
